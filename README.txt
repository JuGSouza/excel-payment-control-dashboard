Excel Payment Control Dashboard

Simple Accounts Payable tracking system built in Microsoft Excel.

This spreadsheet was originally created to help organize and monitor payments for a small business where I previously worked in Brazil. The goal was to provide a clear and simple way to track invoices, payment status, and overdue items.

Features

Structured payment tracking table
Automatic **overdue detection** using Excel formulas
Conditional formatting** for payment status visualization
Pivot table analysis** to summarize payment totals
Simple **dashboard with chart** to visualize payment distribution

Tools Used

Microsoft Excel
Excel Tables
Pivot Tables
Conditional Formatting
Excel formulas (IF, AND, TODAY)
